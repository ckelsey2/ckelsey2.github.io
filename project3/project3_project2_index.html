iPAD LEARNING GAME

The purpose of this project was to create an interactive learning or memory tool, meant for iPad use, using HTML, CSS, jQuery, and JavaScript. I combined animation functions with CSS with toggle script options with jQuery. All assets were created by Photoshop with matte-layers, photo-filters, brushes, and more. Coding completely created by me, with Brackets and Google Chrome Inspector, utilizing a few plug-ins for action.
Tea and health are both personal passions, so I decided to educate users about both, using a very intuitive “click to pour” button system. After several stages of problem solving and user-testing, I decided to utilize a larger button system for visibility and ease. My mood boards pushed me towards a neutral and natural direction. View my mood boards, as well as other information, here on my tumblr page.
Play my created game here. In this project, I learned more and discovered how to connect CSS animations with jQuery toggle functions. I continued to develop my understanding of web design and all its entities.
